# Kiosk Keyboard Plus Example

This example demonstrates how to use the `kiosk_keyboard_plus` package in a Flutter application.

## Features Demonstrated

- Alphanumeric keyboard layout
- Custom styling and theming
- Integration with text fields
- Submit functionality
- Responsive layout

## Running the Example

1. Make sure you have Flutter installed
2. Navigate to this directory:
   ```bash
   cd example
   ```
3. Get dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Code Overview

The example shows a registration screen with:

- A `KioskKeyboardController` managing the input state
- An alphanumeric keyboard for entering registration codes
- Custom styling matching a kiosk UI design
- A text field that displays the entered value
- Submit functionality with a SnackBar confirmation

See [main.dart](lib/main.dart) for the full implementation.
